# 🪢 Hangman Game

A fun word guessing game built with HTML, CSS and JavaScript.

## Features
- Normal mode with 6 categories (Fruits, Animals, Tech, Nature, Music, Sports)
- Extreme mode with random words fetched from an API
- Responsive design — works on laptop and mobile
- Hangman drawing that builds step by step
- Color coded keyboard (green = correct, red = wrong)

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## How to Play
1. Choose difficulty — Normal or Extreme
2. In Normal mode, pick a category
3. Click letters to guess the hidden word
4. You have 6 wrong guesses before the man is hanged!
